# 📊 E-Commerce Analytics Project (SQL + Power BI)

## 🚀 Project Overview

This project analyzes an e-commerce dataset to uncover insights related to:

* Sales performance
* Customer behavior (RFM analysis)
* Customer retention (cohort analysis)
* Product performance
* Market basket analysis

---

## 🛠️ Tools Used

* **SQL (MySQL)** – Data cleaning & transformation
* **Power BI** – Data modeling & dashboarding
* **DBeaver** – SQL execution & export

## 🔍 Key Insights

* Revenue is concentrated in a few categories.
* Customer retention drops significantly after first purchase.
* High-value customers contribute disproportionately to revenue.
* Weak product bundling opportunities exist.

---

## 📈 Business Recommendations

* Improve customer retention through loyalty programs.
* Focus on high-performing product categories.
* Implement cross-selling strategies.
